📋 Role Summary

For this final project, I served as the sole designer and developer of the Lingua World Language Learning Center website. My responsibilities included:

🎨 UI/UX Design

Designed the full visual identity, color system, typography, and layout across all 5 pages to create a cohesive, professional look.

🏗️ HTML5 Structure

Built semantic, accessible HTML using proper tags: header, nav, main, footer, section, article, figure, fieldset, legend, and ARIA roles.

✨ CSS3 Styling

Applied Flexbox layouts, CSS variables, gradients, box-shadow, border-radius, pseudo-classes (:hover, :nth-child), and responsive media queries.

📱 Responsive Design

Ensured the site works on all screen sizes from mobile phones to large desktop monitors using flexible layouts and media queries.
